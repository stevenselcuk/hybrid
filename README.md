# Start to Developing

```javascript
// Clone it
git clone https://github.com/stevenselcuk/hybrid.git

// enter folder
cd hybrid

// Install deps
yarn

// Seed admin account and other demo material
yarn seed

// For using Nodemon

yarn dev

// or using Webpack

yarn dev:webpack

// That's all
```



# Build

Build with Webpack

`yarn build`

or building with Babel

`yarn build:babel`

# Testing

`yarn test`
